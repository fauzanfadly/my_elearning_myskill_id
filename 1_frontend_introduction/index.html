Website Development Introduction

Buatkan web yang berisi form untuk menginput nama, email, dan alamat. Tambahkan button submit. Ketika user klik submit web akan menampilkan data dari form tersebut di bawahnya. Tambahkan validasi setiap input ketika user klik tombol submit untuk memastikan semua input diisi. Buatlah semenarik mungkin tampilan web tersebut.
